Los archivos contenidos en este repositorio son de uso especifico para facturas electronicas emitidas en Mexico, tanto en formato CFD como CFDI. 

El código es de uso libre, así mismo el jar generado. La funcionalidad esta muy limitada actualmente debido a que así fue el requrimiento de un cliente, pero lo iremos ampliando para extraer las partes que se requieren para llevar una contabilidad de tus facturas por mes y los importes de IVA trasladado que obtuviste. La intencion es que sea un ejercicio previo a tu contabilidad.
